# Example: Search & Trending

Search for skills and get trending results using the SDK.

## Setup

```bash
npm install @openbooklet/sdk
```

## Run

```bash
npx tsx index.ts
```

## What it does

1. Searches for skills matching "code review"
2. Prints the top 5 results
3. Fetches the top 5 trending skills this week

## Output

```
Found 8 skills:
  - code-review-pro (acmecorp)
  - typescript-code-review (devtools)
  - ...

Trending this week:
  - nextjs-seo-aeo-skill-2026 (1204 pulls)
  - typescript-strict-mode (876 pulls)
  - ...
```
