# Example: Basic Skill Fetch

Fetch a skill by name using the SDK.

## Setup

```bash
npm install @openbooklet/sdk
```

## Run

```bash
npx tsx index.ts
```

## What it does

1. Creates an OpenBooklet client
2. Fetches `nextjs-seo-aeo-skill-2026` by name
3. Prints the skill's display name, version, publisher, and a content preview

## Output

```
Skill: Next.js SEO & AEO 2026
Version: 1.0.0
Publisher: OpenBooklet

Content preview:
# Next.js SEO & AEO Best Practices 2026
...
```
