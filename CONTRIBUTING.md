# Contributing to OpenBooklet

Thanks for wanting to help! Here's what you can contribute to:

## What's Open for Contributions

- **CLI** (`cli/`) — new commands, bug fixes, UX improvements
- **SDK** (`sdk/`) — new methods, TypeScript improvements, bug fixes
- **MCP Server** (`mcp-server/`) — new tools, resource improvements

## What's NOT in This Repo

The OpenBooklet platform (website, API, database) is proprietary and not
included here. Bug reports for the platform belong in GitHub Issues.

## How to Contribute

1. Fork this repo
2. Create a branch: `git checkout -b fix/your-fix-name`
3. Make your changes
4. Open a pull request with a clear description

## Code Style

- TypeScript strict mode
- No external dependencies in the SDK (zero-dep policy)
- All new CLI commands need a help string

## Questions?

Open a Discussion or email hello@openbooklet.com
